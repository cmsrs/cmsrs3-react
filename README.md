# cmsRS react client

* create server site from:

<https://github.com/cmsrs/cmsrs3>

* change server url in file
```bash
src/config.js
```

* install dependency

```bash
npm install
```

* start app

```bash
npm start
```

* run tests

```bash
npm test
```

* build app

```bash
npm run build --path admin
cp -r build admin
cp -r admin <server_path>/public
```
